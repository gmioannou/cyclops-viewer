# cyclops-viewer

## How to deploy
- npx cap init CyclopsViewer cy.ac.cut.celestia.cyclopsviewer
- ionic build

### Add platforms
- npx cap add ios
- npx cap add android

### Open IDE to build, run and deploy
- npx cap open ios
- npx cap open android

### Syncing the app with Capacitor
- npx cap copy ios
- npx cap copy android
